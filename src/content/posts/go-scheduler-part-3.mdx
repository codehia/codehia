## Size Classes

Go's allocator divides objects into ~70 size classes. Small allocations (≤32KB) go through a per-P cache (mcache) with no locking. Larger ones go directly to the heap.

## mcache → mcentral → heap

1. **mcache** — per-P, no lock needed
2. **mcentral** — shared, has a lock, refills mcache
3. **mheap** — global heap, manages spans of pages

```go
// stack-allocated — no heap involvement
x := 42

// escapes to heap — compiler escape analysis decides
p := &x
_ = p
```
